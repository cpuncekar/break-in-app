# BrakeCycle

BrakeCycle is a mobile-first web application designed to help automotive technicians perform brake break-in (bedding) procedures in a simple, guided format.

The app provides step-by-step instructions, cycle tracking, countdown timers, and safety reminders to make brake break-in easier and more consistent in real-world shop environments.

---

## Features

### Recommended Break-In Procedure
- 20 cycles
- Accelerate to 20 MPH
- Lightly brake down to 5 MPH
- Coast for 10 seconds
- Repeat
- Avoid complete stops when possible

### Minimum Break-In Procedure
- 10 cycles
- Accelerate to 10 MPH
- Light brake application
- 10 second coast/rest period
- Repeat

---

## Safety Features

Before starting a procedure, the app displays safety reminders including:

- Do not perform on highways
- Use only in parking lots or low traffic roads
- Follow all traffic laws and shop safety procedures
- Avoid complete stops when possible

---

## Guided Mode

BrakeCycle includes:

- Step-by-step instructions
- Current cycle tracking
- Countdown timer
- Pause and reset controls
- Completion summary
- Audio/vibration alerts (where supported)

## Technology

Built using:

- HTML
- CSS
- JavaScript
